# MovieApp
Discover many kinds of movies!

## About the Project

### Prerequisites

1. Get a free API Key at [http://www.omdbapi.com/]

2. Clone the repo

```sh
git clone https://github.com/esravsar/MovieApp.git
```

3. Enter your API in `local.properties`

```JS
API_KEY='ENTER YOUR API'
```
